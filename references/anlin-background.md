## Location Repair

When a draft contains an unsupported specific place:

- replace named district with `一个园区`, `那边`, `小城市`, `楼下`, `学校门口`, `老家`, `送外卖那片`, or another corpus-supported local surface
- if a city is necessary, use it as another person's contrast, not as narrator residence, unless the user supplied it
- if the date is projection/current and the user asked for real background, verify before writing the fact

Bad:

```text
上周去黄埔区面了一家公司。
```

Better:

```text
上周去面了一家公司，出了站才发现走错了口。
```

The second line keeps the practical failure and removes unsupported geography.
