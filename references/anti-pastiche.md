# Anti-Pastiche Rules

This file prevents the skill from becoming a remix of recognizable Anlin fragments.

## Principle

Learn mechanisms, not images.

Do not reuse iconic source combinations such as the same product, same body event, same punchline, same relationship, and same ending motion unless the user supplied that exact background and the text is clearly a new context.

## High-Risk Reuse

Treat these as high-risk if they appear without user-supplied need:

- 211 plus exact high salary plus 5000 games plus delivery
- sunlight smell as burnt mites
- wetness/dampness diagnostic system
- A3.1 assets with the same payment-failure arc
- pain plus crawling to toilet with the same sequence
- "world is a troupe of slackers" style line
- direct reuse of "wind through chest" or other signature endings

The point is not to ban every original noun. The point is to avoid copying a recognizable package.

## Overlap Check

When full corpus is available:

1. Run `compare_anlin_corpus.py`.
2. If nearest 3-gram similarity is unusually high relative to corpus-to-corpus baseline, inspect manually.
3. Search the draft's most distinctive 8-12 character phrases in the corpus.
4. Replace copied image packages with new daily material.

Surface overlap is a plagiarism/pastiche alarm, not a style score.

## Safe Transformation

Instead of copying a known image:

- keep the mechanism
- change the object
- change the social relation
- change the bodily consequence
- change the ending action

Example mechanism: abstract pressure collapses into body.

Unsafe: repeat a known urine/pain/crawling sequence.

Safer: a different body interruption grows from the current state card and changes the direction of the scene.
