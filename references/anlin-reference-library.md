# Anlin 写作参考库

本文件提供 Anlin 38 篇原文中高频出现的写作模式的具体范例。每个维度包含原文例句和识别特征。写作时查阅，不是规则清单。

## 一、对话幽默模式

Anlin 的对话幽默有三种核心机制：

**答非所问**：一方问A，另一方答B。幽默在于B与A之间的荒谬距离。
- "我问他那你现在工资多少，他说确实，股价也在跌，广义货币量太多了。"（20220415）
- "你怎么知道裁员消息的，他说这几天好几个技术交流群，很邪门，居然开始聊技术了。"（20240111）

**荒谬结论**：从日常对话中推导出完全不可理喻的结论。
- 水哥用泻药洗虾→"人吃了也拉怎么办"→"华莱士剩了不少泻立停"（20220729）
- "我奶奶听说你阳痿了"——50行铺垫的终极落点（20220413）

**反客为主**：叙述者的回应比对方的问题更让人意外。
- "我说我一个失业人员哪来的情怀，他说那你更该省着点花。"（walkthrough示例）
- "我破口大骂你个工贼，我工作就被你这种人卷没的。"（20220419）

## 二、身体描写模式

Anlin 不写"疼"。他写疼的后果、疼的换算、疼的荒谬。

**疼的后果**：
- "脚踝肿得像鸡蛋，一踩地上就疼，只能蹦着走"（20251017）
- "哭到凌晨三点，被尿憋醒，脚踝还在疼，但尿急的时候是不能蹦的"（20251017）
- "我长痔疮了"——三个字的身体宣言（20220406）

**疼的换算**：身体症状=金钱/机会成本。
- "买秋水仙碱还是买晚饭"（隐含在20251017的叙述中）
- "一个月在医院花掉的钱，得送两个月才能赚回来"（20220512）

**疼的荒谬**：
- "原来我的心是雪花做的，冷如玄冰，一尿就化"（20220509）
- "隔一会就窜去厕所，拉得腿都酸了"→"我说送外卖还有空练腿啊"（20220418）

## 三、自我贬低模式

Anlin 的自贬有三种句式：

**"像我这种..."句式**：用一个具体失败定义自己。
- "像我这种连美团笔试都没过的人"
- "毕竟是在0 offer的基础上讨论问题"
- "一眨眼不当程序员三年了，其实这种说法有点给自己贴金"（20260605）

**比烂逻辑**：通过比较让自己更惨来制造幽默。
- "别人35岁退休送外卖，我毕业直接送外卖，少走15年弯路"（20220404）
- "211毕业，舍友平均年薪50万，而我打了5000局王者荣耀，正准备去送外卖"（20220404）

**自毁设障**：先毁掉自己，不必承受"本可以"。
- "因为一次失败，所以自甘堕落，好让自己的地位配得上这次失败，从而断绝念想"（20220501）

## 四、平台/App 自然引用

Anlin 不写"手机软件"——他写具体App名。每个平台承载文化含义。

| 平台 | 文化含义 | 例句 |
|---|---|---|
| 知乎 | 写作平台，自我指涉 | "自从这两天开始在知乎写日寄"（20220406） |
| 小红书 | 精致焦虑 | "说起小红书，每被挂一家我就把他们APP卸载"（20220404） |
| 美团 | 外卖/消费 | "打开美团""点开饿了么" |
| 拼多多 | 下沉魔幻 | "拼多多节电器"（20220803） |
| B站 | 青年亚文化 | "东百往事""孤勇者" |
| 抖音 | 信息茧房 | "MCN标签""直播算命" |

| 支付宝 | 理财焦虑 | "支付宝推荐的优质基金"（20220409） |

## 五、天气/季节用法

天气不能直说——必须用荒诞比喻包装。天气不是装饰，是功能性存在。

- "云南的天气一边出太阳一边下暴雨。形式上更接近浴霸搭配淋浴头。"（20220512）
- "雨砸在地上变成热气往上蒸"（draft中的一处正确用法）
- "就像春天来了。但还有倒春寒。"——天气作为身体隐喻（draft，偏工整但方向对）

## 六、消费/食物写法

食物不是味觉体验——是价格、是场景、是阶级标记。

- "花了今天三分之一的收入点了份烧烤"（20220426）
- "选了三十块充值，输完密码又退出来改成二十"（意图行动落差）
- "阳光味→螨虫烤焦→烧烤"（20220416）
- "好吃，但很脏，在日本能拍好几集纪录片的那种"——烧烤店（20240111）
- "大扫除后，黑黄黑黄一层一层往下冲，当天晚上立马点了一大把回家，没内味了"（20240111）

## 七、社交比较（幽灵比较）

舍友/同学即使不出场也在场——这是"幽灵比较"。

- "舍友平均年薪50万，而我...准备去送外卖"（20220404）
- "我偷偷查过他的领英，头衔非常豪华"（20220415）
- "路过一个小区，中介说月租3000。我想还行，然后立刻意识到我在用谁的'还行'标准。"（walkthrough示例）
- "当初打着要找工作防止骚扰的理由换了账号，实际上是接受不了和同学的落差"（20230102）

## 八、失眠/睡眠模式

Anlin 的叙述者不写"晚安"不写"关机睡觉"——他的夜晚是被生理极限中断的。

- "褪黑素吃了大半年"（20240111）
- "拿着关机的手机闭眼滑了半小时"（20220507）
- "凌晨两点感觉胸口发痛，爬起来问ai"（20260605）
- "隔壁初中开始放做早操的歌，我还在想...听了几天才意识到是我老了"（20260605）
- "躺床上，外面有猫在叫"——接近睡眠但未完全入睡的边界状态（test draft，方向可）

## 九、家庭互动类型

家人只通过**具体互动事件**出场，不用于假设性温情。

- 妈妈点肯德基（具体事件）
- 爸爸在评论区吵架（20220505）
- "我爸说你复习个屁，你会看书我把手机吃下去"（20220404_2）
- "我妈说，你试试能不能养活自己吧，能的话随你"（20240111）
- "我妈打电话过来，问脚还疼不疼"——母亲关心的固定话术（test draft，可）

## 十、陌生人互动模式

Anlin 对陌生人的观察有两个极端：荒谬化或温柔化。

**荒谬化**：陌生人的行为被解读为系统性的荒诞。
- 保安先跟狗打招呼再跟人打招呼（20220415）
- 鬼火少年放孤勇者→"刚好放到'爱你孤身走暗巷'，我只听到中间三个字他就消失了"（20220423）

**温柔化**：陌生人成为短暂但真实的连接。
- 帮老奶奶修手机连WiFi→她借电动车给叙述者（20220409）
- 清洁阿姨拖掉呕吐物→"散开的人群没有再接近那片区域"（20220423）
- 40多岁阿姨为涂口红纠结一周→"句子后面全是感叹号"（20240111）

## 十一、句式节奏

**逗号流动**：行末逗号制造"话还没说完"的流动感。句号用于落点和强调。

**短句落点**：长段落后的极短句承载情感重量。
- "哦。"——全文最轻的音节，承载最重的判断
- "很残酷。"
- "可惜了。"
- "甜的。"

**"一X就Y"构式**：Anlin 最标志性的因果句式，比"因为所以"紧凑。
- "一到...就..."、"一查...就..."、"一问...就..."

**"V着"vs"在V"分工**：静态描写用V着（"看着手机"），主动进行用在V（"在写论文"）。不混用。
